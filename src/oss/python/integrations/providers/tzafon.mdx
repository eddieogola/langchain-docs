[Tzafon](https://docs.tzafon.ai/overview) is a platform for reliable document loading and web extraction.

## Installation and Setup

-   Get an API key from [tzafon.ai](https://tzafon.ai) and set it in environment variables (`TZAFON_API_KEY`).
-   Install the integration package:

<CodeGroup>
```bash pip
pip install langchain-tzafon
```
```bash uv
uv add langchain-tzafon
```
</CodeGroup>

## Document loader

See a [usage example](/oss/integrations/document_loaders/tzafon).

```python
from langchain_tzafon import TzafonLoader
```
